D-day 위젯 사용 가이드

D-day 위젯은 블로그, 카페, 웹사이트에 삽입하여 목표일까지 며칠 남았는지 실시간으로 표시할 수 있는 무료 위젯입니다. 시험일, 생일, 기념일 등 중요한 날짜를 시각적으로 표현하여 방문자들에게 동기부여를 제공합니다.

위젯이란?

D-day 위젯은 작은 HTML 코드 조각으로, 다른 웹사이트에 삽입하면 해당 페이지에 D-day 카운트다운이 표시됩니다. 위젯은 매일 자동으로 업데이트되어 정확한 남은 일수를 보여줍니다.

위젯 사용 방법

1단계: 위젯 코드 생성

메인 페이지의 "블로그용 위젯 코드 생성" 섹션에서 다음 정보를 입력하세요:

  • 위젯에 표시할 날짜: 목표일을 선택합니다
  • 위젯 제목: 위젯 상단에 표시될 제목을 입력합니다 (예: "시험일까지", "생일까지")

입력 후 "위젯 코드 생성" 버튼을 클릭하면 HTML 코드가 생성됩니다.

2단계: 위젯 코드 복사

생성된 HTML 코드를 복사합니다. 코드는 다음과 같은 형태입니다: 

<iframe src="https://ashlight.store/widget.html?date=2025-12-25&title=시험일까지" 
        width="300" 
        height="150" 
        frameborder="0" 
        scrolling="no">
</iframe>

3단계: 블로그/웹사이트에 삽입

복사한 코드를 원하는 위치에 붙여넣습니다:

  • 네이버 블로그: HTML 편집 모드에서 원하는 위치에 삽입
  • 티스토리: HTML/CSS 편집에서 사이드바나 본문에 삽입
  • 워드프레스: HTML 블록이나 위젯 영역에 삽입
  • 카페/커뮤니티: HTML 편집이 가능한 게시판에 삽입
  • 자체 웹사이트: HTML 소스 코드의 원하는 위치에 삽입

위젯 파라미터 설명

위젯 URL에는 다음 파라미터를 사용할 수 있습니다:

  • date (필수): 목표일 (형식: YYYY-MM-DD, 예: 2025-12-25)
  • title (선택): 위젯 제목 (기본값: "D-day")

예시 URL:

https://ashlight.store/widget.html?date=2025-12-25&title=시험일까지

위젯 크기 조정

위젯의 크기는 iframe의 width와 height 속성으로 조정할 수 있습니다:

  • 작은 위젯: width="250" height="120"
  • 중간 위젯: width="300" height="150" (기본값)
  • 큰 위젯: width="400" height="200"

반응형 디자인을 원하시면 width="100%"를 사용하되, 컨테이너의 최대 너비를 CSS로 제한하는 것을 권장합니다.

위젯 활용 사례

📚 학생 블로그

수능일, 시험일까지의 D-day를 블로그에 표시하여 동기부여를 받고, 방문자들에게도 목표를 공유할 수 있습니다.

🎂 개인 블로그

생일, 기념일, 여행일까지의 카운트다운을 표시하여 특별한 날을 기다리는 설렘을 표현할 수 있습니다.

💼 비즈니스 웹사이트

프로젝트 마감일, 이벤트 개최일, 제품 출시일 등을 표시하여 고객들에게 중요한 일정을 알릴 수 있습니다.

🏃 운동/다이어트 블로그

목표 달성일, 대회일까지의 D-day를 표시하여 지속적인 동기부여를 받을 수 있습니다.

위젯 특징

  • 자동 업데이트: 매일 자동으로 남은 일수가 업데이트됩니다
  • 반응형 디자인: 다양한 화면 크기에 자동으로 적응합니다
  • 아름다운 디자인: 그라데이션 배경과 깔끔한 레이아웃
  • 커스터마이징: 제목과 날짜를 자유롭게 설정 가능
  • 완전 무료: 회원가입이나 결제 없이 무료로 사용 가능

주의사항

  • 위젯은 iframe을 사용하므로, 일부 웹사이트에서는 보안 정책에 의해 차단될 수 있습니다
  • 위젯의 날짜는 서버 시간(한국 시간)을 기준으로 계산됩니다
  • 과거 날짜를 입력하면 "D+숫자" 형식으로 표시됩니다
  • 위젯은 HTTPS 환경에서 가장 잘 작동합니다

문제 해결

위젯이 표시되지 않는 경우:

  • HTML 코드가 올바르게 복사되었는지 확인하세요
  • 날짜 형식이 YYYY-MM-DD인지 확인하세요
  • 웹사이트가 iframe 사용을 허용하는지 확인하세요
  • 브라우저 콘솔에서 오류 메시지를 확인하세요
