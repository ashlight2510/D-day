# D-day 계산기

목표일까지 며칠 남았는지 쉽고 빠르게 계산하는 웹 애플리케이션입니다.

## 주요 기능

### 1. D-day 계산
- 목표 날짜를 입력하여 며칠 남았는지 계산
- 이벤트 이름을 지정하여 저장 가능
- D-day, D+ (과거 날짜) 자동 계산

### 2. 역계산 기능
- N일 후가 언제인지 계산
- 시작 날짜를 지정하여 정확한 날짜 계산

### 3. 빠른 계산
- 50일, 100일, 200일, 1년 후 날짜를 한 번에 계산
- 자주 사용하는 기간을 빠르게 확인

### 4. 날짜 저장 및 관리
- 중요한 날짜를 로컬 스토리지에 저장
- 저장된 날짜 목록에서 D-day 확인
- 저장된 날짜 삭제 및 재계산 기능

## 기술 스택

- HTML5
- CSS3 (반응형 디자인)
- Vanilla JavaScript
- LocalStorage API
- Google AdSense

## 사용 방법

1. 목표 날짜를 선택하거나 입력
2. 이벤트 이름 입력 (선택사항)
3. "계산하기" 버튼 클릭
4. 결과 확인 및 저장

## 배포

이 웹사이트는 정적 파일로 구성되어 있어 어떤 웹 호스팅 서비스에서도 배포할 수 있습니다.

### 배포 시 주의사항

1. `ads.txt` 파일을 웹사이트 루트 디렉토리에 배치해야 합니다.
2. Google AdSense 승인 후 광고가 표시됩니다.
3. HTTPS를 사용하는 것이 좋습니다 (AdSense 요구사항).
